Unit 2: Introduction to CSS

1. What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and design web pages. CSS controls the layout, colors, fonts, spacing, and overall appearance of a website.

2. Why Use CSS?

  • Makes webpages look attractive
  • Separates content from design
  • Saves time by applying styles to multiple pages
  • Improves user experience

3. Types of CSS

Inline CSS

CSS applied directly inside an HTML tag.

<p style="color:red;">This is a paragraph</p>

Internal CSS

CSS written inside the <style> tag in the head section.

<style>
p{
color:blue;
}
</style>

External CSS

CSS written in a separate file and linked to HTML.

<link rel="stylesheet" href="style.css">

4. CSS Syntax

selector {
property: value;
}
Example: 
h1{
color: blue;
font-size: 30px;
}

5. Advantages of CSS

  • Improves website design
  • Reduces code repetition
  • Makes websites responsive
  • Easy to maintain
