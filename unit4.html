Unit 4: HTML Forms and Tables

1. HTML Forms

HTML forms are used to collect user input. The information entered by users can be sent to a server for processing. Forms are commonly used for login pages, registration forms, and surveys.

2. Form Tag

The <form> tag is used to create a form.

<form>
Name: <input type="text">
</form>

3. Common Input Types

  • Text – For entering text
  • Password – For entering passwords
  • Email – For email addresses
  • Radio – For selecting one option
  • Checkbox – For selecting multiple options
  • Submit – To submit the form

4. Example Form

<form>
Name: <input type="text"><br>
Password: <input type="password"><br>
<input type="submit" value="Submit">
</form>

5. HTML Tables

Tables are used to display data in rows and columns.

6. Table Tags

  • <table> – Creates a table
  • <tr> – Table row
  • <th> – Table heading
  • <td> – Table data

7. Example Table

<table border="1">
<tr>
<th>Name</th>
<th>Age</th>
</tr>

<tr>
<td>John</td>
<td>20</td>
</tr>
</table>
