Unit 3: HTML Lists and Images

1. HTML Lists

HTML lists are used to organize information in a structured way. There are mainly two types of lists in HTML.

Unordered List

This list displays items with bullet points.

<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>
Example:
  • HTML
  • CSS
  • JavaScript

Ordered List

This list displays items with numbers.

<ol>
<li>Open Notepad</li>
<li>Write HTML Code</li>
<li>Save as .html file</li>
</ol>
Example:
  1. Open Notepad
  2. Write HTML Code
  3. Save the file

2. HTML Images

Images can be added to a webpage using the <img> tag. 

<img src="image.jpg" width="300">
Explanation:
  • src – Specifies the image path
  • width – Controls the image width

3. Advantages

  • Lists help organize content clearly
  • Images make webpages more attractive
  • Improves user understanding
